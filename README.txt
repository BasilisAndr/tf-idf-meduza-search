Основной файл - crawler.py.
Две основные функции - prepare() для сбора корпуса, создания обратного индекса и в целом предобработки, main() для собственно поиска.
prepare() уже запущена, всё необходимое находится в корне и папке src.
При запуске файла crawler.py запустится функция main(), которая будет искать по собранному корпусу новостей с медузы.
